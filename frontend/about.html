About Image Forgery Detection

Project Overview

This application is designed to detect image tampering and forgery using advanced machine learning techniques. It combines Convolutional Neural Networks (CNN) with Error Level Analysis (ELA) to identify manipulated images with high accuracy.

Our goal is to provide a reliable tool for detecting digital image forgeries, which has become increasingly important in an era where image manipulation is common and easily accessible.

Technologies Used

  • Convolutional Neural Networks (CNN): Deep learning architecture specialized for image analysis
  • Error Level Analysis (ELA): A forensic method to identify areas within a digital image that have different compression levels
  • Electron: Cross-platform desktop application framework
  • TensorFlow: Open-source machine learning framework
  • Python: For backend processing and analysis
  • SQLite: For user authentication and data storage

How It Works

Our application works through a two-step process:

  1. Error Level Analysis (ELA): First, we generate an ELA representation of the image which highlights differences in JPEG compression quality levels. Modifications to an image often leave traces that are visible in the ELA.
  2. CNN Classification: The ELA representation is then processed through our trained CNN model, which has learned to recognize patterns characteristic of both authentic and tampered images.

The result includes a classification (authentic or tampered) along with a confidence score indicating the model's certainty in its assessment.
